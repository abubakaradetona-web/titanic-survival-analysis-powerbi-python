## 🚢 Titanic Survival Analysis

### 📌 Project Overview
This project explores the Titanic dataset to uncover the key factors that influenced passenger survival. The analysis combines Python for data cleaning and Power BI for data modeling and visualization.

The goal was to transform raw, unstructured data into meaningful insights through a structured analytical workflow.

---

### 🎯 Objectives
- Clean and prepare the dataset for analysis  
- Identify patterns influencing survival outcomes  
- Build an interactive dashboard for data exploration  
- Communicate insights clearly using visual storytelling  

---

### 🛠 Tools & Technologies
- **Python (Pandas, NumPy)** – Data cleaning and preprocessing  
- **Power BI** – Data modeling, calculated columns, and dashboard design  

---

### 🧹 Data Preparation (Python)
- Handled missing values in key columns (Age, Cabin, Embarked)  
- Converted data types for consistency  
- Standardized column names and formats  
- Ensured data quality for accurate analysis  

---

### 📊 Data Modeling (Power BI)
Developed calculated columns to enhance analysis:
- **Age Group** (Child, Teen, Adult, Senior)  
- **Fare Group** (Low, Medium, High, Very High)  
- **Family Size** (SibSp + Parch + 1)  

Created key measures:
- Total Passengers  
- Survival Count  
- Death Count  

---

### 📈 Dashboard Features
- KPI cards showing total passengers, survivors, and deaths  
- Survival analysis by gender, class, age group, and fare group  
- Interactive filters for dynamic exploration  
- Clean and structured visual layout for clarity  

---

### 🔍 Key Insights
- Gender had a strong influence on survival, with females having higher survival rates  
- Passenger class significantly impacted survival probability  
- Higher ticket fares were associated with increased chances of survival  
- Age and family structure influenced survival outcomes  

---

### 💡 Impact
This project demonstrates the ability to:
- Work across multiple tools in a data workflow  
- Clean and prepare real-world datasets  
- Build dashboards that communicate insights effectively  
- Apply analytical thinking to uncover meaningful patterns  

---

### 🚀 Conclusion
The analysis highlights how socio-economic factors and demographics influenced survival during the Titanic disaster. It also showcases the power of data analytics in uncovering insights from historical data.
